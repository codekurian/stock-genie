üìä Stock Data APIs

GET Get Recent Stock Data

/api/v1/stocks/{symbol}?days={days}

Description: Fetches recent stock data for a given symbol (currently returns mock data)

Parameters:

  • symbol - Stock symbol (e.g., AAPL, MSFT, GOOGL)
  • days - Number of days (default: 30)
curl "http://localhost:8080/api/v1/stocks/AAPL?days=7"

GET Get Latest Stock Data

/api/v1/stocks/{symbol}/latest

Description: Returns the latest available date for a stock symbol

curl "http://localhost:8080/api/v1/stocks/AAPL/latest"

POST Refresh Stock Data

/api/v1/stocks/{symbol}/refresh

Description: Requests a refresh of stock data for the given symbol

curl -X POST "http://localhost:8080/api/v1/stocks/AAPL/refresh"

üìà Technical Analysis APIs

GET Get Technical Analysis

/api/v1/analysis/{symbol}/technical?days={days}&indicators={indicators}

Description: Returns available technical indicators (implementation in progress)

Parameters:

  • symbol - Stock symbol
  • days - Number of days (default: 30)
  • indicators - Comma-separated list of indicators (optional)
curl "http://localhost:8080/api/v1/analysis/AAPL/technical?days=7"

GET Get Available Indicators

/api/v1/analysis/{symbol}/indicators

Description: Returns list of available technical indicators

curl "http://localhost:8080/api/v1/analysis/AAPL/indicators"

POST Calculate Technical Indicators

/api/v1/analysis/{symbol}/calculate

Description: Calculates specified technical indicators (implementation in progress)

curl -X POST "http://localhost:8080/api/v1/analysis/AAPL/calculate" \ -H "Content-Type: application/json" \ -d '{ "days": 7, "indicators": ["SMA", "RSI"] }'
