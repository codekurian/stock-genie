üìä Stock Data APIs

GET Get Recent Stock Data

/api/v1/stocks/{symbol}?days={days}

Description: Fetches recent stock data with real Alpha Vantage API integration. Automatically falls back to mock data if API key not available or rate limits exceeded.

Parameters:

  • symbol - Stock symbol (e.g., AAPL, MSFT, GOOGL)
  • days - Number of days (default: 30, max: 365)

Features: Real API integration, rate limiting, database caching, error handling

curl "http://localhost:8080/api/v1/stocks/AAPL?days=7"

GET Get Latest Stock Data

/api/v1/stocks/{symbol}/latest

Description: Returns the latest available stock data for a symbol. Fetches from database first, then API if needed.

curl "http://localhost:8080/api/v1/stocks/AAPL/latest"

POST Refresh Stock Data

/api/v1/stocks/{symbol}/refresh

Description: Forces a refresh of stock data by clearing cache and fetching fresh data from API.

curl -X POST "http://localhost:8080/api/v1/stocks/AAPL/refresh"

üìà Technical Analysis APIs

GET Get Technical Analysis

/api/v1/analysis/{symbol}/technical?days={days}&indicators={indicators}

Description: Returns technical analysis with real indicator calculations (SMA, EMA, RSI, MACD, OBV).

Parameters:

  • symbol - Stock symbol
  • days - Number of days (default: 30)
  • indicators - Comma-separated list of indicators (optional)

Available Indicators: SMA_20, SMA_50, EMA_12, EMA_26, RSI_14, MACD, OBV

curl "http://localhost:8080/api/v1/analysis/AAPL/technical?days=7"

GET Get Available Indicators

/api/v1/analysis/{symbol}/indicators

Description: Returns list of available technical indicators with descriptions.

curl "http://localhost:8080/api/v1/analysis/AAPL/indicators"

POST Calculate Technical Indicators

/api/v1/analysis/{symbol}/calculate

Description: Calculates specified technical indicators with custom parameters.

curl -X POST "http://localhost:8080/api/v1/analysis/AAPL/calculate" \ -H "Content-Type: application/json" \ -d '{ "days": 7, "indicators": ["SMA_20", "RSI_14", "MACD"] }'

ü§ñ AI Analysis APIs

GET Check LLM Status

/api/v1/llm/status

Description: Checks local LLM service status and availability. Shows if Ollama is running and ready.

curl "http://localhost:8080/api/v1/llm/status"

POST Test LLM

/api/v1/llm/test

Description: Tests the LLM service with a custom prompt. Returns AI-generated response.

curl -X POST "http://localhost:8080/api/v1/llm/test" \ -H "Content-Type: application/json" \ -d '{ "prompt": "Hello, how are you?" }'

POST Analyze Stock Data

/api/v1/llm/analyze

Description: Comprehensive AI stock analysis using local LLM. Provides buy/sell/hold recommendations with reasoning.

curl -X POST "http://localhost:8080/api/v1/llm/analyze" \ -H "Content-Type: application/json" \ -d '{ "symbol": "AAPL", "analysisType": "stock-analysis", "stockData": [] }'
