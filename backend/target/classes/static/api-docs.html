üìà Stock Data

GET /api/v1/stocks/{symbol}

Get historical stock data for a symbol
symbol (path) - Stock symbol (e.g., AAPL)
days (query, optional) - Number of days (default: 30)

GET /api/v1/stocks/{symbol}/latest

Get the latest stock data for a symbol
symbol (path) - Stock symbol (e.g., AAPL)

POST /api/v1/stocks/{symbol}/refresh

Refresh stock data from external API
symbol (path) - Stock symbol (e.g., AAPL)

üìä Technical Analysis

GET /api/v1/analysis/{symbol}/indicators

Get available technical indicators
symbol (path) - Stock symbol (e.g., AAPL)

GET /api/v1/analysis/{symbol}/technical

Calculate technical indicators for a stock
symbol (path) - Stock symbol (e.g., AAPL)
days (query, optional) - Number of days (default: 30)
indicators (query, optional) - Comma-separated list of indicators

POST /api/v1/analysis/{symbol}/calculate

Calculate specific technical indicators
symbol (path) - Stock symbol (e.g., AAPL)
days (body) - Number of days (default: 30)
indicators (body) - Array of indicator names
Request Body Example:
{
  "days": 14,
  "indicators": ["RSI_14", "SMA_20", "MACD"]
}
